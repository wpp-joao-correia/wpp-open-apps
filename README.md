# wpp-open-apps

A repository for testing applications within the WPP Open Platform.

## Purpose

This repository serves as a testing ground for applications that are being developed or integrated with the WPP Open Platform. It allows developers to:

- Test new application features in a controlled environment
- Validate application behaviour before production deployment
- Share test applications with the team for collaborative testing

## Getting Started

### Prerequisites

- Node.js (recommended version: 18.x or later)
- Yarn package manager
- Access to the WPP Open Platform environment

### Installation

```bash
# Install dependencies
yarn install

# Start the development server
yarn start
```
